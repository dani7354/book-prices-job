## Testing
Change to test directory:\
`cd BookPricesJob.Test/`

Run all:\
`dotnet test`

See all tests available:\
`dotnet test --list-tests`

Run filtered:\
`dotnet test  --filter "FullyQualifiedName~BookPricesJob.Test.IntegrationTest"`
